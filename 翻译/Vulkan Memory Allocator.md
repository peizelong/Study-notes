### 快速使用

#### **项目设置**

Vulkan Memory Allocator是一个只有一个头文件的stb风格的库。不需要将它构建为一个单独的库。可以直接将文件添加到项目中，并将其提交到代码存储库中，与其他源文件放在一起。
